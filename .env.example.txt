# Google Safe Browsing API Configuration
# Para obter sua chave API:
# 1. Acesse https://console.developers.google.com/
# 2. Crie um novo projeto ou selecione um existente
# 3. Ative a API "Safe Browsing API"
# 4. Crie credenciais (API Key)
# 5. Copie a chave e cole abaixo

VITE_GOOGLE_SAFE_BROWSING_API_KEY=sua_chave_api_aqui

# Exemplo de arquivo .env.local
# VITE_GOOGLE_SAFE_BROWSING_API_KEY=AIzaSyB...

# Observações importantes:
# - A API tem limite gratuito de 10.000 consultas por dia
# - Mantenha a chave em segurança e não compartilhe publicamente
# - Para produção, considere usar restrições de IP na Google Cloud Console
